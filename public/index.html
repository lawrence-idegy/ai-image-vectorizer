What is a Vector File?

A vector file uses mathematical formulas to define shapes, making it infinitely scalable without losing quality. Unlike pixel-based images, vectors maintain crisp, clean edges at any size.

Why Use Vectors?

  • Scalability: Scale up or down infinitely without quality loss
  • Smaller file size: Generally more efficient than raster images
  • Editability: Easy to modify colors, shapes, and sizes
  • Professional output: Standard for logos, branding, and printing

When to Use Each Method

AI Vectorization

Best for:

  • Colored logos and illustrations
  • Complex designs with gradients
  • Photographs (advanced)
  • Professional quality output

Potrace (Fallback)

Best for:

  • Simple black & white images
  • Line art and sketches
  • Icons and basic shapes
  • Quick local processing

Understanding Quality Metrics

  • TRUE VECTOR: Contains only mathematical paths (scales infinitely)
  • Quality Score: 90-100 = Excellent, 70-89 = Good, 50-69 = Fair
  • Vector Elements: Number of geometric shapes in the SVG
  • Complexity: Simple to Very Complex based on detail level

Advanced Features

  • Background Removal: Remove background before vectorization for cleaner results
  • Detail Level: Control how much detail is preserved (Low/Medium/High)
  • Export Formats: SVG (web), PDF (print), EPS (legacy), AI (Illustrator)

Tips for Best Results

  • Use high-resolution source images (500px+)
  • Ensure good contrast in your original image
  • Use AI method for colored/complex images
  • Remove backgrounds for cleaner vectorization
  • Choose appropriate detail level for your use case
